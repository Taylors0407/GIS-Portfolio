## Projects

### Human Trafficking Vulnerability Analysis
Spatial analysis of factors contributing to human trafficking vulnerability in the Pittsburgh Metropolitan Statistical Area.

### Blue Spruce Park Map Revision Project
GIS and UAV mapping project focused on updating park maps, trails, amenities, and spatial data for Blue Spruce Park in collaboration with the Indiana County Planning Department.

### Saxis Shoreline Extraction
Python and ArcPy workflow for shoreline delineation and change detection.

### Land Use / Land Cover Change Detection
Analysis of land cover changes in Las Vegas using NLCD datasets.

### Crime Data PCA Analysis
Principal Component Analysis of state crime statistics using R.
