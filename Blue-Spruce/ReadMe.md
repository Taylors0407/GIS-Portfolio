Blue Spruce Park Map Revision Project

Completed as part of Indiana University of Pennsylvania's UAV Applications course, this project involved collaborating with classmates 
to update and improve maps of Blue Spruce Park in Indiana County, Pennsylvania. Working with the Indiana County Planning Department as 
the project stakeholder, the team integrated UAV imagery, GPS field data, DEMs, trails, park amenities, and aerial photography to create 
more accurate and up-to-date park maps.

The project provided hands-on experience in geospatial data collection, UAV operations, cartographic design, and spatial data management. 
Deliverables included updated park maps, drone orthomosaics, elevation visualizations, and land cover classifications to support park 
planning and recreation management.

Tools Used: ArcGIS Pro, UAV Mapping, GPS Data Collection, Orthomosaics, DEM Analysis, Cartography, PASDA Data
